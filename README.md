# tpre-python

本项目是全国密码学竞赛设计项目，该项目是使用python实现tpre算法

## 项目原理  

使用国密算法实现分布式代理重加密tpre

## 环境依赖

该项目依赖以下软件：  
python 3.11
gmssl
gmssl-python

## 安装步骤

```
pip install
```
